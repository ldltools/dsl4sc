# Installation on Debian/Ubuntu
## Prerequisites

- [flow_parser]()  
  `opam install wtf8 flow_parser`

- [escodegen]()  
  `npm install -g escodegen`

- [flow]() (optional)  
  `npm install flow`

- [esparse]() (optional)  
  `npm install esparse`
